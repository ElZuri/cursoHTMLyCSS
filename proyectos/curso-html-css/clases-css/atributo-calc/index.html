Atributo CALC.

Descripción.

Tambien cabe mencionar que para el tamaño de las cajas se puede usar el atributo calc, el cual tiene la siguiente sintaxis.

width: calc(40% - 100px)

Como se muestra en este texto que tiene una proporcion del 40% de la pantalla menos 100px, termina siendo una caja algo pequeña, pero nos puede ser util en ocaciones especificas.
