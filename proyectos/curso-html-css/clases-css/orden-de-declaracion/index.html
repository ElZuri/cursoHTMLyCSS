Especificidad en CSS3.

Orden de declaración.

El orden en el que el navegador aplica los estilos es el siguiente:

  1. Importancia.
  2. Especificidad.
  3. Orden de

  • Importancia.

    • Lista de orden de importancia:

    1. Hoja de estilo de agente de usuario (Estilos de Navegador).
    2. Declaraciones normales en hojas de estilo de autor (Nuestro CSS).
    3. Declaraciones importantes en hojas de estilos de autor (!important).

  • Especificidad.

    • La especificidad es un numero de importancia que le da el navegador a los selectores para escoger un estilo.

    En esta el valor es mas importante de izquierda a derecha.

    Tambien existe un valor llamado "!important" con el que le decimos al navegador que este estilo es el se debe escoger sobre todos los demas, aunque esto es una mala practica ya que hay muchos proyectos con este atributo repetido multples veces.

  • Reglas de Cascada

    • Los estilos CSS3 son elegidos por el navegador como una cascada, de ahi su nombre, en el cual el navegador realiza un proseso de arriba hacia abajo y selecciona el ultimo escalon que cumplio con la condición.

    Orden de las fuentes.

    En tus propios estilos, las declaraciones al final del documento, anularan a las que sucedan antes en caso de conflicto.

    Como se muestra en la imagen anterior, las etiquetas body y section establecen un color para el texto diferente. Pero como section esta al final del ducumento, anula el estilo de body.

    Esto solo funciona cuando tienen la misma especificidad, ya que ambos son etiquetas y tienen 0,0,1 de especidad.

    Nota*: Especificidad en Visual Studio Code.

    En Visual Studio Code podemos ver la especificidad de los selectores con poner el ratón encima.

    Como se puede apreciar tengo un estilo tipo id, el cual es de una especidad mayor a la de etiqueta, haciendo que el siguiente texto que pondre con la id "codigo" tendra el color verde, aunque la etiqueta body y bisque esten pordebajo de el, ya que esta propiedad de color tienen menor especificidad para el navegador.
