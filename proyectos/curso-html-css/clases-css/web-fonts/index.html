Clase: Web Fonts.

Podemos insertar fuentes externas a nuestros estilos CSS, pero lo recomendable es usar un solo tipo de estilo para todo un proyecto ya que esto le da una identidad mas clara de este mismo, pues usar varias fuentes genera es una mala practica y un mal diseño.

En esta pagina usamos una fuente llamada Horror-Sketch en la barra nav, la cual fue puesta en una carpeta independiete llamda fonts y enlazada a la pagina en nuestro documento CSS con el siguiente comando.

@font-face {

font-family: Horror-Sketch;

src: url(../fonts/Horror-Sketch.ttf);

}
