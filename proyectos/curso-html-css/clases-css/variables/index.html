Clase: Variables

Podemos crear variables con los valores de atributos que queramos utilizar mas de una vez para tener mayor control de estos mismos.

Las variables se puenden crear en cada selector que querramos usarlas.

Ejemplo

Si creamos una varible con el color bisque en body como se muestra acontinuacion pero un color rojo por defecto.

body {

--primary-color: bisque;

color:red;

}

Y creamos una un selector main con el color de la variable.

main {

color: var(--primary-color);

}

El resultado seria que toda la platilla body tendria color rojo y los contenedores main un color bisque como este documento.

Variables en esta plantilla.

Para efectos mas visuales, cree variables para cada color y tamaño de fuente que pense que debia tener separarada para poder modificarla a mi gusto.

Y como quiero que estas variables sean utilizadas den todo la platilla CSS, las declare en el selector :root{}.

:root{

/*cabecera*/

--header-size: 3rem;

--navbar-background-color: #075255;

--links-color: blanchedalmond;

--links-hover-color: burlywood;

--links-active-color: rgb(226, 145, 40);

--link-background-color: rgb(43, 10, 10);

/*main*/

--primary-color: bisque;

--secundary-color: burlywood;

--title-color: chocolate;

--subtitle-primary-color: rgb(255, 201, 6);

--subtitle-secundary-color: rgb(115, 100, 76);

--main-border-set: 0.3px solid #ccc;

--font: 2rem;

--back-color: black;

--p-margin-top: 1rem;

/*Codigo*/

--code-font-family: Lucida Console, "Courier New", monospace;

--code-color: rgb(5, 255, 5);

--code-background-color: rgb(9, 9, 52);

/*Footer*/

--footer-background-color: rgb(22, 2, 41);

}
