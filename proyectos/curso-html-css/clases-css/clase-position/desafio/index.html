Clase: Position

La propiedad position de CSS especifíca cómo un elemento es posicionado en el documento. Las propiedades top, right, left y bottom determinan la ubicacion final de los objetos.

Tipos de posicionamiento

Un elemento posicionado es un elemento cuyo valor computado de position es relative, absolute, fixed o sticky. Osea cualquiera menos static.

Relative

Las propiedades de posicion ubican al elemento en relacion a su posición actual estatica.

Absolute-Fixed

Las propiedades de posición especifícan el desplazamiento desde los bordes del contenedor padre y los demas elementos estáticos de ignoran el elemento y su anterior posicion static.

Fixed se alinia al contendor padre principal.

Sticky

Es tratado como relative hasta que el contener padre deja la ventana y este se queda pegado a la ventana con una distancia establecida por las propiedades de posición.
