Clase: Imagenes Responsive.

Las imágenes adaptables son una parte importante del diseño web responsivo. El objetivo es hacer que las imágenes funcionen bien en dispositivos con una amplia diferencia de tamaño de pantallas, resoluciones y otras características. Esto ayuda a mejorar el rendimiento en diferentes dispositivos. HTML proporciona herramientas para ayudar a implementar imágenes adaptables, como srcset y el elemento.

Para hacer que una imagen sea responsiva, tienes que dar un nuevo valor a su propiedad de anchura. La altura de la imagen se ajustará automáticamente. Lo importante es saber que siempre debes utilizar unidades relativas para la propiedad de anchura, como el porcentaje, en lugar de las absolutas, como los píxeles.
