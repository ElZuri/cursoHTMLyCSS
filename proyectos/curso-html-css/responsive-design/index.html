Clase: Media Queries.

Es la forma en que podemos optimizar los estilos para varios tipos de pantalla como mobile, tablet y pc.

Mobile Firts

Sirve para hacer diseños principalmente para mobile, luego los crecemos a tablet y luego a pc.

Tambien existe Only mobile, donde solo se diseña para mobile.

media queries en CSS

Para establecer estilos por el tamaño de pantalla, usamos el siguiente codigo.

@media (min-width:480px) {

...

}

@media (min-width:768px) {

...

}

@media (min-width:1024px) {

...

}

En la primer sección del código, tenemos el estilo para celulares, el segundo para tablets y el tercero para pc.

Siguiendo la regla de cascade, primero el navegador cargara los estilos para celular, despues cargara los estilos para tablet y por ultimo los estilos para pc.

Pero esto generaria un css muy grande el cual el navegador descargaria cada vez que se habra la página.

para mejorar eso usaremos:

media queries en HEAD

Con este método, creamos tres archivos css, uno llamada mobile-style.css, tablet-style.css y pc-style.css. y despues de eso, los vinvularemos a nuestro HTML en el head de la siguiente forma.

<link href="mobile-style.css" rel="stylesheet">

<link href="tablet-style.css" rel="stylesheet" media="screen and (min-width:768px)">

<link href="pc-style.css" rel="stylesheet" media="screen and (min-width:1024px)">

Así el navegador solo cargará el que ocupe dependiendo del tamaño de la pantalla.

Esta es una buena práctica.
