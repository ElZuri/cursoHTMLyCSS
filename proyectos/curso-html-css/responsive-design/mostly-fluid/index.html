Clase: Mostly fluid.

Consiste en una cuadricula fluida que se reacomoda según el ancho de la pantalla, manteniendo el mismo tamaño del contenido y ajustando sus márgenes.

Este patrón solo necesita un punto de interrupción entre las pequeñas y grandes pantallas, y se puede combinar con la técnica de Column Drop.

Este página tiene estilos especiales para el tamaño de la fuente, caja main e imagenes de logo del header.
