labels, alt y title.

Son atributos HTML que se utilizan para mejorar la accesibilidad y la usabilidad de una página web.

Label

El elemento <label> representa una etiqueta para un elemento en una interfaz de usuario. Este puede estar asociado con un control ya sea mediante la utilización del atributo for, o ubicando el control dentro del elemento label. Tal control es llamado “el control etiquetado” del elemento label

Alt

El atributo <alt> se utiliza para proporcionar un texto alternativo para imágenes y otros elementos no textuales. Este texto se muestra en caso de que la imagen no pueda cargarse, y también es utilizado por lectores de pantalla para describir el contenido de la imagen a usuarios con discapacidades visuales

Title

El atributo title se utiliza para proporcionar información adicional sobre un elemento. Este texto se muestra como un “tooltip” cuando el usuario pasa el cursor sobre el elemento. También puede ser utilizado por lectores de pantalla para proporcionar información adicional a usuarios con discapacidades visuales.

Estos atributos son importantes para mejorar la accesibilidad y la usabilidad de una página web, y deben utilizarse adecuadamente para proporcionar una experiencia de usuario óptima.
