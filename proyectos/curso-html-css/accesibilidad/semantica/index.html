La semántica en HTML

La semántica en HTML se refiere al uso de etiquetas HTML para reforzar el significado de la información en las páginas web, en lugar de simplemente definir su presentación. Las etiquetas semánticas describen claramente su contenido tanto para el navegador como para el desarrollador. Por ejemplo, en lugar de utilizar etiquetas `<div>` con atributos `id` o `class` para indicar la navegación, el encabezado y el pie de página, se pueden utilizar elementos semánticos como `<nav>`, `<header>` y `<footer>`.

El uso de elementos semánticos ayuda a mejorar la accesibilidad y la optimización de motores de búsqueda, ya que proporciona una estructura clara y significativa para el contenido de la página. También puede hacer que el código sea más fácil de leer y mantener para los desarrolladores.
